# Semi_Supervised_Lesion_Classification
***Final Project of HKUST 2023 Spring ELEC4010N - Artificial Intelligence for Medical Image Analysis***

Implementing ResNet with semi-supervised learning to classify whether the lesion dataset is benign or malignant.

Please read the [slides](https://github.com/dnvkjn/Semi_Supervised_Lesion_Classification/blob/main/final_project_slides.pdf) and [report](https://docs.google.com/document/d/1trjqGPizzLC-guVhTlJHwhUH9jqv_DEOpftEOAZTeWg/edit) for more details of the project.
